We store data on your computer

If you login: We store your access token we retrieve from github on your computer as a cookie. See the Code

This is done so that we can access the github api without data restrictions.
This is purely optional and we never use your user information.
This is off by default.
In order to activate, click the login button below when not logged in.
In order to deactivate, click the logout button below when logged in.

By Using the Blog: We store data we get from github on your available local storage See the Code

This is done so that you may avoid as many calls to github as possible.
Without this, you would have to request github every time you wanted to view a page.
This will speed up your page load times and ensure you don't have to login
This is on by default.
By deactivating, you will also delete all previously cached information.
In order to deactivate, click the Don't Cache button below.
In order to activate, click the Cache Results button below.
